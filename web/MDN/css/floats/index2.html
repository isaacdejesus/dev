Simple float example

Float

Float works by removing floated element out of the normal layout flow of the doc. In this case, we float the div to the left of the parent element, (the body). Content under the div in normal layout now wrap around the div, filling up space to the right all the way to the top of the floated element. Since floated element is taken out of flow, the boxes after are behind the float

Since floated items are removed from normal flow and items following it will display besides it and behind it. we can prevent elements from moving up, we use the clear function. Clear takes following values: left, right, both. This paragraph is using clear left, to prevent moving p up to the left.

clear function doesn't work if items of different sizes are inside the same container. instead of clear, we use display: flow-foot;
